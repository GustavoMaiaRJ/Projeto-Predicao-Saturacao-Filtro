# projeto-predicaosaturacaofiltro
Este projeto desenvolve um modelo numérico para suprir a falta de monitoramento preditivo de uma ETA compacta. O código calcula o instante exato de saturação ($t^*$) do filtro rápido de areia — que colapsa aos 3,0 MCA devido à argila amazônica — e o respectivo volume de água tratado. O diferencial da abordagem é a simulação da dinâmica real da planta, utilizando vazão oscilante e considerando as especificidades das quatro estações hidrológicas do ano.
